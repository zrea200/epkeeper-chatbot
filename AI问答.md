### 售电业务二十问

此篇就客户关于售电业务咨询频率最高的若干问题作解答：

1.售电到底是什么？

自2015年3月颁布电改9号文至今，电改已进行7年多。这次改革叫做“管住中间，放开两头”。

管住中间：就是在电网、输配电环节强化政府管理，因为这是自然垄断环节。形象地讲，输配电网相当于高速公路，只要向电网交了过路费，发电企业和用电企业就可以逐步实现直接交易。

放开两头：就是对发电方、售电方、用电方提高市场化程度，实现更加充分的竞争。

售电公司作为发电方和用电方连接的桥梁，一手牵着电厂批发买电，一手牵着电力用户零售卖电。售电公司因为交易规模大，与电厂的议价能力强，能帮用户买到更便宜的电源。

2.什么是电力市场化？

电力市场化即还原电力的商品属性，基于市场经济原则，发电厂和电力用户本着公平竞争、自愿互利的原则，通过协商、竞价等方式，就电能及其相关产品进行交易，通过市场竞争确定价格和数量。

3.电力用户为什么要参与电力市场化交易？

建立电力市场旨在通过开放、竞争等市场手段实现电力能源资源的优化配置。电力市场化交易让电力用户可以直接与发电厂或售电公司议价，获得发电企业传递的让价。参与电力市场化交易，是电力用户降低用电成本的有效途径。

4.哪些电力用户可以参与市场交易？

《国家发展改革委关于进一步深化燃煤发电上网电价市场化改革的通知》（发改价格【2021】1439号）中明确指出：推动工商业用户都进入市场。

10千伏及以上用户原则上要全部直接参与市场交易，其他工商业用户也要尽快进入市场，暂未直接从电力市场购电的用户由电网企业代理购电。

我们可以看到工商业用户全部进入市场是必然趋势，电网企业代理购电只是过渡性政策。

5.电力用户直接参与市场交易有什么渠道？

一种是直接向发电企业购电，另一种是通过售电公司购电。

6.如何入市？

未入市的电力用户可在每季度最后15日前，在电力交易平台完成市场注册程序，选择下一季度起直接参与市场交易。

7.注册入市需要提交什么资料？

企业营业执照复印件（盖企业公章）；

企业法人身份证复印件（盖企业公章）；

高压供用电合同复印件；

供用电补充协议（盖企业公章、国网电力公司公章、骑缝章）：至所辖供电公司营业厅办理，每个户号四份；

直购电知悉书（盖企业公章）；

授权委托书（盖企业公章）；

银行开户许可证（盖企业公章）。

8.已入市用户如果退出市场会有什么结果？

已直接参与市场交易在无正当理由情况下改由电网企业代理购电的用户，用电价格=电网企业代理购电价格的1.5倍+输配电价+政府性基金及附加。  
已直接参与市场交易的高耗能用户，不得退出市场交易。

9.交易窗口（交易周期）有哪些？

目前上海电力直接交易有三个交易窗口：

年度交易：一般在前一年的年末组织开展，交易标的为下一年全年各月各时段电量。

月度交易：一般在前一月的中旬开展，交易标的为下月各时段电量。

月内交易：在当月开展，交易标的为当月范围内各时段电量。

10.交易方式有哪些？

1）双边协商——售方申报，购方确认

2）集中竞价——集中竞价，边际出清

3）挂牌交易——目前主要用于电网企业代理购电方式

4）滚动撮合——购售双方滚动申报，滚动撮合交易出清

上海目前年度、月度及月内的交易方式主要为双边协商和集中竞价。

11.交易结算周期？

按月、分时段（峰平谷时段）结算，月结月清。

12.用电价格最终由几部分组成？

直接交易用户的用电价格由五部分组成，在市场交易电价的基础上，叠加输配电价、政府性基金及附加、辅助服务费用、以及电力保障综合费用。

13.鉴于工商业目录电价取消，峰、平、谷各时段电价如何确定？

上一问所说的电力用户的用电价格则确定了平时段的价格，以平时段价格为基础，峰、谷时段按照比例上浮和下浮。

平时段电价=用电价格

峰时段电价=平时段电价\*（1+上浮比例）

谷时段电价=平时段电价\*（1-下浮比例）

上下浮比例按政府主管部门的规定执行。

14.上海市分时电价的时段是如何划分的？

2023年1月1日起上海市工商业用户执行新的分时电价，时段划分如下图：

有4个月份新增了尖峰时段，每年7月和8月12:00-14:00 的2个小时为尖峰时段；1月和12月19:00-21:00的2个小时为尖峰时段。

此外，以前夏季（7、8、9月）的11:00-13:00为平时段，现在改为高峰时段或尖峰时段，也就是说夏季的峰时段延长了2个小时。

比价水平也完全按照新的标准执行，如下图：

夏季（7、8、9月）和冬季（1、12月）的高峰时段在平时段电价基础上上浮80%，普遍比之前上涨了20%；尖峰时段在平时段电价基础上上浮125%，上浮比例比之前增加了65%；这几个月的用电成本会相应上涨。

15.如何通过电管家购电？

用户与电管家签订双方协议（上海电力交易中心提供的合同范本），双方约定合同套餐，并上传交易中心平台进行合同备案，完成零售关系绑定。

16.什么是市场批发用户和市场零售用户？

入市用户直接与发电企业签订合同购电，则是批发用户。

通过售电公司购电的用户则是零售用户。

17.成为市场用户后，是发电厂直接接线到企业吗？会对日常用电产生影响吗？

用户参与电力交易签订的是金融合约，并不是直接将电点对点的从电厂直接送到用户，电厂发的电统一是由电网进行调度的，电力输配、电网建设、运维调度，全还是由电网负责。对原有用电质量、付费方式不会产生任何影响。

18.偏差考核是什么？

入市用户是需要对自己每月的用电量进行提前预估申报的，基于申报电量来进行市场交易，当实际用电量与市场交易电量有差异时，便产生了偏差，根据交易规则对偏差电量执行一个处罚性的罚款，这就是偏差考核。

偏差分为正偏差和负偏差，都是需要考核的。正偏差是指用户实际用电量超过市场交易电量；负偏差是指用户实际用电量小于市场交易电量。

19.偏差考核怎么计算？

按照偏差电量占市场合同电量的比例，分为四级偏差考核：

一级(≤±x%)：免偏差考核

二级（±x%~±10%）: 基准价的1%(415.5\*1%=4.16)

三级（±10%~±20%）：基准价的2%（415.5\*2%=8.31）

四级（＞±20%）：基准价的3%（415.5\*3%=12.47）

(7、8、9月份x=4,其他月份x=2，基准价为415.5元/兆瓦时)

举例1：某用户6月份结算单：

该用户6月申报并交易853兆瓦时，实际用量1187兆瓦时，超用334兆瓦时，正偏差39%，偏差考核费用3024元，占当月电费的0.33%。

举例2：某用户9月份结算单：

该用户9月申报并交易1251兆瓦时，实际用量816兆瓦时，少用435兆瓦时，负偏差35%，偏差考核费用3656元，占当月电费的0.55%。

20.通常说的498.6元是什么价格？

根据沪发改价管【2021】51号文，上海的市场交易在现行基准价415.5元/兆瓦时的基础上上下浮动，原则上不超过20%。

498.6则为415.5上浮20%的市场交易封顶价。

### 绿电业务问答

1.绿电是什么？

绿电即绿色电力，是指符合国家有关政策要求的风电、光伏等可再生能源发电企业上网电量。市场初期，主要是指风电和光伏发电企业上网电量。根据国家有关要求可逐步扩大至符合条件的其它电源上网电量。

2.绿电交易的概念？

绿色电力交易是指以绿色电力产品为标的物的电力中长期交易，交易电力同时提供国家规定的可再生能源绿色电力证书（以下简称“绿证”），用以满足发电企业、售电公司、电力用户等市场主体出售、购买绿色电力产品的需求。

3.绿色电力证书是什么？

绿色电力证书是国家对发电企业每兆瓦时可再生能源上网电量颁发的具有唯一代码标识的电子凭证，作为绿色电力环境价值的唯一凭证

4.绿电交易规则有哪些？

绿色电力环境价值随绿色电力交易由发电企业转移至电力用户，绿色电力环境价值应确保唯一，不得重复计算或出售。

绿色电力交易价格应充分体现绿色电力的电能价值和环境价值，原则上市场主体应分别明确电能量价格与绿色环境权益价格。

5.如何成为绿电用户？

如果企业需要购买绿电，企业必须先成为市场化用户，通过上海市电力交易中心进行注册，入市购买绿电。

6.如何开展绿电交易？

参与绿色电力交易的售电公司购买绿色电力产品，通过电力零售合同销售给有绿色电力消费需求的用户。

7.电管家集团股份有限公司绿电交易情况？

2023年电管家集团股份有限公司在北京电力交易平台通过集中竞价的方式为售电用户购买了56.1万度绿电，计算绿色环境权益电价（0.02849/kWh）的平均价格为0.5071/kWh。

8.绿电结算的结算与市场化用户结算的差异？

绿电用户结算与非绿电用户结算形式上不存在差异，都是按照每月国网上海电力公司提供的电费账单进行缴费。

绿电用户与非绿电用户结算的费用中是存在一定差异的，非绿电用户电费结算当月使用次月结算，绿电用户的绿电费用当月结算，但是绿色环境价值需要隔月进行结算，这是按照《北京电力交易中心绿色电力交易实施细则（修订稿）>的通知》（京电交市〔2023〕44号），绿色电力交易电能量与绿色电力环境价值是分开结算的。即本月的电费账单的绿电结算价格不含本次绿电的绿电环境权益电价，该笔费用会在下月进行补缴。

### 用户侧储能20问

什么是用户侧储能？

我们平时说的用户侧储能主要是指应用在广大工商业客户的电化学储能，储能设备可简单理解为一个大型的充电宝。在电价低时充电，在电价高时放电用于企业生产，从而降低企业用电成本。

安装储能系统的好处有哪些？

（1）削峰填谷：利用峰谷电价差，在谷期和平期充电，在峰期和尖峰期放电，降低企业用电成本。

（2）平衡需量电费：储能系统可以进行削峰填谷，消除尖峰负荷，平滑用电曲线，减少需量电费。

（3）动态增容：用户的变压器容量是固定的，一般情况下当用户需要变压器在某个时段超负荷运，则需要进行变压器扩容。而安装匹配的储能系统后，则可以在该时段通过储能放电，降低变压器负荷，从而减少变压器增容改造费用。

（4）需求侧响应：安装储能系统后，如果电网发出需求响应，客户可以不用限电、也不用缴纳该时段的高昂电费，反而有可能通过储能系统参与需求响应交易，获取额外的补偿费。

用户侧储能主要应用场景？

用户侧储能主要应用于充电站、工业园区、数据中心、通信基站等用电规律的对象。

关于用户侧储能的投资建设运营收益模式？

（1）由集成运营商进行投资、建设、运营，客户不需出钱，只需提供场地供储能系统建设使用，所获取的收益由集成运营商与用户共享。

（2）双方共享收益，收益分成根据客户用电情况、储能规模大小进行协商。

（3）由用户自行投资建设储能系统，集成运营商负责运营，并承诺客户收益。

用户侧储能项目的合作年限？

建议的合作年限一般为15年。

从合同签约到项目落地，需要多长时间？

从签约到项目正式投运，一般需要3~6个月，视场地具体情况和复杂程度而定。

安装储能电站需要企业达到什么用电标准？

目前国家、地方政府没有对用户侧储能建设设置条件。一般来说，用电量越大，项目可配置的储能容量越大。

储能电站的占地面积是多少？

1MWh的储能电站占地面积为10㎡左右，如考虑安全距离则为20-30㎡。

安装储能电站对场地要求有哪些?

(1) 场地位于室外，且20米内无危险化学品仓库。

(2) 尽量靠近配电房，距离配电房最远不宜超过 100 米。

储能电站的施工周期多久？是否需要断电？要断多久？

手续办妥后，正式施工周期约1-1.5个月；需要短时间断电，用于安装并网柜，最短断电时间约2个小时。

储能电站完工后谁来负责运营、维护？是否需要业主方人员参与？

对于选择某个集成运营商的客户来说，储能电站完工后将由集成运营商负责运营、维护，同时一般会给业主方相关工作人员做简单培训，并提供在特殊情况下的处理方案。

储能电站是否会增加业主方的基本电费？

不会。如果是按容量计费的，基本电费本就是固定的。如果是按需量计费的，则还有可能因为储能放电承担了负荷，降低了最大需量从而降低基本电费。在进行储能容量设计时，已考虑过变压器的容量与负载，不会因为储能充电去拉高需量，如果会拉高需量时，能量管理系统（EMS）会自动判别并停止储能充电。

储能电站是否会产生损耗？

若产生损耗由哪方负责？有损耗，在双方进行收益分成时已将损耗考虑在内并扣除。

储能电站计量是否准确？

准确，安装的所有电费计量表都是采用符合国家标准要求并通过检测的计量表。

储能电站发生安全问题由谁负责？

因乙方设备导致甲方设备故障或损毁灭失的，甲方因此遭受的损失由乙方赔偿。同时，乙方在合同履行期间负责投保与项目有关的机器损坏险和财产一切险等，并承担保险费，财产保险投保数额不少于投保财产的实际价值。

储能电站日常运行时，用户是否需要断电？

电站日常运行不需要用户断电。

储能电站切换时会不会造成电力闪烁？

不会，储能设备不会对用户电能质量产生负面影响。

储能电站进行紧急供电时是瞬时接入还是需要手动调整？

需要提前设置切换策略，储能设备与UPS不同，无法做到全自动切换。

储能设备达一定程度损耗后是否会更换，产生费用由谁承担？

储能设备的主要损耗部分是电芯，电芯的健康度会随着使用时间的增加而下降，下降到一定程度后集成运营商会负责更换，费用无需企业承担。

### 上海地区电力需求响应常见问题解答

什么是电力需求响应？

电力需求响应是指电力市场价格明显升高（降低）或系统安全可靠性存在风险时，电力用户根据价格信号或激励措施，暂时调整其固有的习惯用电模式，减少（增加）用电，从而促进电力供需平衡、保障系统稳定运行的行为。

为什么要进行需求响应？

实施需求响应可以有效调节用电峰谷负荷，缓解供需矛盾，保证电力系统的平稳运行。

需求响应的分类？

根据电网运行需要，电力需求响应可以分为削峰和填谷两种。

削峰是指在电力需求高峰期，通过调整用户的电力消费行为来降低峰值负荷，从而减少电力系统的负荷压力。这种需求响应主要针对电力系统在供电压力大的情况下，需要通过减少用户用电量来避免系统超负荷运行，以保证电力系统的稳定性。比如，在夏季高温天气下，人们普遍使用空调，导致电力需求激增，此时需要通过需求响应手段来削减用电峰值。

填谷则是指在电力需求低谷期，通过调整用户的电力消费行为来增加谷底负荷，从而提高电力系统的利用效率。这种需求响应主要针对电力系统在供电压力小的情况下，需要通过增加用户用电量来提高电力系统的利用率。比如，在夜间低谷期，人们往往使用电力设备较少，此时可以通过电力价格梯度等方式来鼓励用户增加用电量，以提高电力系统的利用率。

相较于传统调峰手段的有序用电而言，需求响应有什么优势？

|     |     |     |
| --- | --- | --- |
| 比较项 | 有序用电 | 需求响应 |
| 负荷调节量 | 由政府评估，电力用户比较被动 | 由电力用户根据自身情况自主申报 |
| 强制性 | 针对每次负荷调节任务，必须完成 | 针对每次负荷调节邀约，可根据实际情况决定是否进行响应 |
| 奖惩  | 完成负荷调节任务无奖励，未完成有惩罚 | 完成响应有经济补偿，未响应或响应不成功无惩罚 |
| 时长  | 单次实施时间长，一般在半天以上 | 单次实施时间短，一般在半小时至两小时左右 |

哪些负荷资源可以有效调节？

传统高载能工业负荷、工商业可中断负荷、用户侧储能、电动汽车充电设施、分布式发电、智慧用电设施等。

准入条件

电力用户应具有上海市内独立的电力营销户号。

具备一定的负荷调节能力，响应能力小于1000千瓦的用户原则上应通过负荷集成商（虚拟电厂）参与响应。

如何参与？

首先，电力用户应在上海市需求侧响应管理平台（以下简称需求响应平台）完成注册并通过审核；通过负荷集成商参与响应的由负荷集成商代为注册。

当满足需求响应启动条件时，上海市电力需求响应中心根据国网上海市电力公司确定的响应指标信息，发布响应邀约。电力用户、负荷集成商在规定时间内通过需求响应平台反馈是否参与本次需求响应，以及意向的响应容量，逾时未反馈的视为不参与。

响应效果如何评定？

通过需求响应平台监测、自动记录并判断需求响应实施效果。

计量依据供电企业专业计量器具，以电网企业用电信息采集系统数据为准。

补贴如何结算？

采用次清年结方式，国网上海市电力公司应于响应后的30日内，完成补贴计算工作。用户年度响应补贴按年度确定与公示，并通过电费账单减项体现。

电管家如何帮助用户做好电网的需求响应工作？

电管家作为负荷聚合商，帮助用户注册、上报负荷资源、及时传导和解释政策，并组织用户实施邀约，指导用户梳理可调节的负荷，提供全方位的技术支持。

### 虚拟电厂相关问题解答

什么是虚拟电厂？

“虚拟”意味着并非实体，“虚拟电厂”并不是真正意义上的发电厂，而是一种基于能源互联网技术的“看不见的电厂”。

它是一种将不同空间的楼宇空调、充电站、分布式能源、非连续性生产、储能等一种或多种资源聚合起来，实现自主协调优化控制，起到提供电能、调峰调频等电厂所具有的作用，参与电力系统运行和电力市场交易的智慧能源系统。

为什么要推广虚拟电厂？

新型电力系统对灵活性资源的需求是催生虚拟电厂的根本原因。

虚拟电厂的各个可调节资源可自由组合，容量可大可小，并且响应更快。这种灵活性，在参与新型电力系统调峰、调频或调压等调节中非常重要。

并且虚拟电厂相比传统调节手段如火电厂、水电厂、抽水蓄能等显示出优越的经济性，因此成为新型电力系统建设的重要方向。

目前虚拟电厂政策有哪些？

从2021年开始，国家发改委和能源局频频发布关于虚拟电厂的政策文件，为虚拟电厂的持续发展奠定了基础。

2023年发布的两大核心政策文件，《电力需求侧管理办法 (2023版)》和《电力现货市场基本规则 (试行)》，要求推动建立和完善需求侧资源与电力运行调节的衔接机制，逐步将需求侧资源以虚拟电厂等方式纳入电力平衡，重点推进新型储能、虚拟电厂、车网互动、微电网等技术的创新和应用，推动分布式发电、负荷聚合商、储能和虚拟电厂等新型经营主体参与交易。

2024年6月1日起施行的《电力市场监管办法》明确新增虚拟电厂作为电力交易主体，这将为可控负荷、新型储能、分布式新能源等灵活性资源提供进入市场的机会。

具有独立的电力营销户号，并具有负荷调节能力的用户都可以参与。
